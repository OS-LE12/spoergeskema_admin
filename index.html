Del med brugere

Download bruger-fil

Filen indeholder spørgeskemaet indlejret som data. Brugere åbner den i en browser og besvarer spørgsmålene. Deres svar gemmes i filen — eller du kan modtage dem via e-mail (se nedenfor).

Sådan virker det

1. Klik Download bruger-HTML og upload som index.html til dit bruger-repository
2. Del QR-koden eller linket med dine brugere
3. Brugeren åbner spørgeskemaet og besvarer det
4. Første bruger bliver bedt om at indtaste Dispatch Token én gang i deres browser
5. Svar sendes automatisk til GitHub Actions og gemmes i data.json
6. Åbn admin og klik ↓ Hent fra GitHub for at se svarene

⚠ Brugerne skal indtaste Dispatch Token én gang. Del tokenet med dem på forhånd via e-mail.
