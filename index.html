Del admin-adgang

Generer en adgangsfil til en kollega. De åbner filen i browseren og viderestilles automatisk til admin med alle indstillinger klar.

Sådan deler du filen

1. Download brugerfilen og upload den til GitHub
2. Del QR-kode eller URL med brugerne
3. Svar sendes automatisk til GitHub — åbn admin og klik ↓ Hent for at se dem
